# Lessons Learned

프로젝트 전반에 걸쳐 누적된 교훈. 단일 스킬을 넘어 팀 전체가 알아야 할 패턴.

> 단순 증상-해결 패턴은 각 `skills/*/SKILL.md`에 기록.
> 여기에는 설계 수준의 결정, 여러 레이어에 영향을 미치는 패턴만 기록.

---

## 작성 방법

`/writing-skills` 스킬이 아래 형식으로 자동 추가:

```markdown
### [YYYY-MM-DD] [주제]

**배경**: 어떤 상황에서 발생했는지

**교훈**: 무엇을 배웠는지

**적용 범위**: 어떤 스킬/레이어에 영향을 미치는지

**관련 failure-log**: FL-NNN
```
